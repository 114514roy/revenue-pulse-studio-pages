Revenue Pulse Studio Support

What Revenue Pulse Studio Does

Revenue Pulse Studio turns a local sales export into a weekly revenue pulse and Markdown report. It is built for indie mobile developers who want a quick weekly readout without rebuilding a spreadsheet.

Supported Import Format

Import or paste CSV/TSV text with these columns:

  • Date
  • SKU
  • Title
  • Units
  • Proceeds
  • Currency of Proceeds (optional; defaults to USD when omitted)

The parser also accepts common developer sales report aliases:

  • Begin Date
  • Product SKU
  • App Name
  • Units Sold
  • Developer Proceeds
  • Currency of Proceeds

Purchase And Restore

Markdown export is a $9.99 one-time purchase. It is not a subscription. Use Restore purchase if you reinstall the app or move to another device with the same purchase account.

Troubleshooting

  • If import fails, confirm the file is UTF-8 or UTF-16 text.
  • If the pulse looks empty, confirm the export includes at least two weeks of rows.
  • If a currency error appears, import one proceeds currency at a time.
  • If export is locked after purchase, tap Restore purchase.
  • If the file exporter closes without saving, choose a local Files destination and try again.
  • Use Save locally only when you want this device to remember the current full report text.
  • Use Clear local report to remove stored report text from this device.
  • Use Load sample to restore the bundled demo report.

Privacy

The current release runs locally, does not require an account, does not use a server, and does not connect to external developer portals.
