Revenue Pulse Studio Support

What Revenue Pulse Studio Does

Revenue Pulse Studio turns a local App Store sales export into a weekly revenue pulse and Markdown report. It is built for indie iOS developers who want a quick weekly readout without rebuilding a spreadsheet.

Supported Import Format

Import or paste CSV/TSV text with these columns:

  • Date
  • SKU
  • Title
  • Units
  • Proceeds
  • Currency of Proceeds (optional; defaults to USD when omitted)

The parser also accepts common App Store Connect-style aliases.

Purchase And Restore

Markdown export is a $9.99 one-time purchase. It is not a subscription. Use Restore Purchase if you reinstall the app or move to another device using the same Apple ID.

Troubleshooting

  • If import fails, confirm the file is UTF-8 or UTF-16 text.
  • If the pulse looks empty, confirm the export includes at least two weeks of rows.
  • If a currency error appears, import one proceeds currency at a time.
  • If export is locked after purchase, tap Restore Purchase.

Privacy

The current release runs locally, does not require an account, does not use a server, and does not connect to App Store Connect.
